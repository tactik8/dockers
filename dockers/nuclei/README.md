## How to use
1. Deploy nuclei on docker node using compose
2. Run nuclei
3. Let nuclei update its plugins
4. Run nuclei using command line:
```
docker exec -it nuclei-scanner nuclei -u https://XYZ.com -o /root/nuclei-output/scan_results_XYZ.txt
```
5. Results will be shown in terminal. Copy/paste to gemini and ask to generate report.
